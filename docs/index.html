Eight-node brick element (C3D8)

The C3D8 element is a general purpose linear brick element, fully integrated (2x2x2 integration points). The shape functions can be found in [36]. The node numbering follows the convention of Figure hex8 and the integration points are numbered according to Figure hex8gp. This latter information is important since element variables printed with the *EL PRINT keyword are given in the integration points.
